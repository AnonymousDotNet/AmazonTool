# AmaTool.Wpf - WPF 客户端

本项目为 AmaTool 的 WPF 桌面客户端，提供现代化的 UI 与自动化功能操作入口。

## 功能简介

- 可视化任务管理界面
- 支持插件扩展和脚本运行
- 任务执行日志查看
- 与后端 API 实时交互

## 环境依赖

- .NET 8.0 或以上
- Windows 10/11

## 快速运行

1. 使用 Visual Studio 打开解决方案
2. 设置 `AmaTool.Wpf` 为启动项目
3. 直接运行（F5）或发布为独立 EXE

## 配置

- 配置文件：`appsettings.json`
  - API 地址、主题设置等可在此文件自定义

## 常见问题

- 启动失败：请确认已安装 .NET 8.0 环境
- 无法连接 API：请检查后端服务是否已启动
